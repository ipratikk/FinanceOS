# CoreML Export Pipeline

## Purpose

Define the complete CoreML export pipeline for all 9 CoreML models in the FinanceIntelligence platform. Every model must be exportable from source via Python scripts with no manual GUI steps. This document specifies the export script structure, coremltools usage patterns, metadata requirements, quantization strategy, artifact naming, hash computation, and CI integration.

---

## Non-Negotiable Constraints

1. Every model artifact (`.mlpackage`) must be reproducible from `training/<model>/export_coreml.py`
2. No model may be created via Xcode, CreateML GUI, or CoreML Model Editor
3. `coremltools` is the only permitted export mechanism
4. All artifacts are generated outputs; Python source is the source of truth
5. Artifacts are not committed to git; they are generated by CI and stored in artifact store

---

## Export Script Template

Each model follows this structure:

```
training/<model_name>/
├── train.py              ← training
├── evaluate.py           ← evaluation
├── export_coreml.py      ← CoreML export  ←── THIS DOCUMENT
├── benchmark.py          ← benchmarking
├── requirements.txt      ← Python dependencies
└── README.md
```

### NLModel Export (Models 1, 2, 3, 6)

```python
# training/category/export_coreml.py

import coremltools as ct
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

def export_nlmodel(model_path: str, output_path: str, label_map: dict, version: str):
    # Load fine-tuned transformer
    tokenizer = AutoTokenizer.from_pretrained(model_path)
    torch_model = AutoModelForSequenceClassification.from_pretrained(model_path)
    torch_model.eval()

    # Trace with example input
    example_input = tokenizer(
        "UPI-ZEPTO MARKETPLACE Zepto upi",
        return_tensors="pt",
        max_length=128,
        truncation=True,
        padding="max_length"
    )

    traced_model = torch.jit.trace(
        torch_model,
        (example_input["input_ids"], example_input["attention_mask"])
    )

    # Convert to CoreML
    coreml_model = ct.convert(
        traced_model,
        inputs=[
            ct.TensorType(name="input_ids", shape=(1, 128), dtype=int),
            ct.TensorType(name="attention_mask", shape=(1, 128), dtype=int),
        ],
        outputs=[ct.TensorType(name="logits", dtype=float)],
        compute_units=ct.ComputeUnit.CPU_AND_NE,  # include Neural Engine
        minimum_deployment_target=ct.target.iOS17,
    )

    # Add NLModel-compatible metadata
    coreml_model.input_description["input"] = "Transaction narration text"
    coreml_model.output_description["classLabel"] = "Predicted category"
    coreml_model.output_description["classLabel_probs"] = "Confidence scores per class"
    
    coreml_model.author = "FinanceOS Intelligence Team"
    coreml_model.version = version
    coreml_model.short_description = "Transaction Category Classifier"
    
    # Add class labels
    spec = coreml_model.get_spec()
    spec.description.metadata.userDefined["label_map"] = json.dumps(label_map)
    spec.description.metadata.userDefined["dataset_version"] = DATASET_VERSION
    spec.description.metadata.userDefined["training_commit"] = GIT_COMMIT

    # Quantize (8-bit weight quantization)
    config = cto.OptimizationConfig(
        global_config=cto.OpLinearQuantizerConfig(mode="linear_symmetric", dtype=torch.int8)
    )
    coreml_model = cto.linear_quantize_weights(coreml_model, config=config)

    # Save
    output_file = Path(output_path) / f"TransactionCategoryClassifier_v{version}.mlpackage"
    coreml_model.save(str(output_file))
    
    # Compute + print hash
    artifact_hash = compute_directory_hash(output_file)
    print(f"Artifact SHA256: {artifact_hash}")
    print(f"Saved to: {output_file}")
    
    return artifact_hash
```

### Embedding Model Export (Model 7)

```python
# training/embedding/export_coreml.py

import coremltools as ct
from sentence_transformers import SentenceTransformer
import torch

def export_embedding_model(model_path: str, output_path: str, version: str):
    model = SentenceTransformer(model_path)
    model.eval()

    # Wrap to output fixed-size embedding
    class EmbeddingWrapper(torch.nn.Module):
        def __init__(self, model):
            super().__init__()
            self.model = model
        
        def forward(self, input_ids, attention_mask):
            outputs = self.model(input_ids=input_ids, attention_mask=attention_mask)
            # Mean pooling
            token_embeddings = outputs.last_hidden_state
            mask_expanded = attention_mask.unsqueeze(-1).float()
            embeddings = (token_embeddings * mask_expanded).sum(1) / mask_expanded.sum(1)
            # L2 normalize
            return torch.nn.functional.normalize(embeddings, p=2, dim=1)

    wrapper = EmbeddingWrapper(model._modules["0"].auto_model)
    wrapper.eval()

    # Trace
    dummy_ids = torch.zeros(1, 64, dtype=torch.long)
    dummy_mask = torch.ones(1, 64, dtype=torch.long)
    traced = torch.jit.trace(wrapper, (dummy_ids, dummy_mask))

    coreml_model = ct.convert(
        traced,
        inputs=[
            ct.TensorType(name="input_ids", shape=(1, 64), dtype=int),
            ct.TensorType(name="attention_mask", shape=(1, 64), dtype=int),
        ],
        outputs=[ct.TensorType(name="embedding", dtype=float)],
        compute_units=ct.ComputeUnit.CPU_AND_NE,
        minimum_deployment_target=ct.target.iOS17,
    )

    # Float16 weights (embedding models are sensitive to int8 quantization)
    coreml_model = ct.compression_utils.affine_quantize_weights(
        coreml_model, nbits=16
    )

    output_file = Path(output_path) / f"NarrationEmbedder_v{version}.mlpackage"
    coreml_model.save(str(output_file))
    artifact_hash = compute_directory_hash(output_file)
    print(f"Artifact SHA256: {artifact_hash}")
    return artifact_hash
```

## Hash Computation

```python
# training/scripts/compute_artifact_hash.py

def compute_directory_hash(path: Path) -> str:
    """
    Deterministic SHA256 of all files in a directory tree.
    Files sorted by relative path to ensure reproducibility.
    """
    h = hashlib.sha256()
    for f in sorted(path.rglob("*")):
        if f.is_file():
            h.update(str(f.relative_to(path)).encode())  # include path in hash
            h.update(f.read_bytes())
    return h.hexdigest()
```

## Artifact Storage

Artifacts are NOT committed to git. Storage strategy:

- **Development:** `training/artifacts/` (local, gitignored)
- **CI artifacts:** GitHub Actions artifact store (30-day retention)
- **Production:** Embedded in app bundle via Xcode build phase

## Quantization Strategy

| Model Type | Quantization | Rationale |
|---|---|---|
| NLModel classifiers (1, 2, 3, 6) | int8 linear symmetric | Good accuracy/size tradeoff for text classifiers |
| Tabular models (4, 9) | None needed | Already tiny (< 1 MB) |
| Embedding model (7) | float16 | Embedding distances sensitive to int8 quantization |
| Anomaly detector (9) | int8 | Tabular features robust to quantization |

## Risks

| Risk | Mitigation |
|---|---|
| coremltools version incompatibility with PyTorch version | Pin both in `requirements.txt`; test export in CI environment |
| Traced model captures wrong input shapes | Use `ct.TensorType` with explicit shapes; test with variable-length input |
| Quantization degrades accuracy past threshold | Evaluate quantized model on golden set before registering; roll back if F1 drops > 0.01 |
| App bundle size exceeds App Store OTA update limit (25 MB for cellular) | Models embedded in main bundle download; ensure initial download flow documented |
